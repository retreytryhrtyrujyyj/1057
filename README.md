# 网上花店销售系统

## 项目简介
网上花店是基于Java Web技术/SSM框架/JSP页面技术/Mysql数据库的的在线鲜花销售系统。 功能上支持： 1.登录/注册 2.鲜花产品详情 3.购物车 4.下单购买 5.个人信息管理 等功能模块。 适合作为毕业设计、大作业、课程设计。
